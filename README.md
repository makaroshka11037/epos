# epos
Попытка создания бота для получения дз из ЭПОС ПЕРМЬ, неудача из-за токена который сбрасывается раз в шесть часов.


## настройка
если вдруг вы готовы менять токен и перезапускать бота раз в пол дня, то...
Откройте код, вставьте все нужные значения в переменные ОЧЕНЬ ВНИМАТЕЛЬНО ЧИТАЯ ПОДСКАЗКИ.
Все данные по запросу на ЭПОС можем найти в логе хттп запросов при входе на страницу с расписанием и дз.
